# 💻 Fundamentos de Programação em C#: Gestão de Memória e Controles

Este repositório contém exercícios práticos desenvolvidos durante a graduação em ADS, focados no entendimento profundo de como o C# manipula dados e interfaces.

## 🧠 Conceitos Aplicados:
* **Passagem por Valor (`value`):** Onde uma cópia do dado é enviada para a função, preservando o valor original na memória.
* **Passagem por Referência (`ref`):** Onde o endereço de memória é compartilhado, permitindo que a função altere diretamente a variável original.
* **Manipulação de Controles de UI:** Lógica para limpeza e reset de componentes visuais em Windows Forms/WPF.

## 🛠️ Tecnologias
* Linguagem: **C#**
* IDE: **Visual Studio 2022**
* Framework: **.NET** (Programação Visual)
